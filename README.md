# SVA (Super Verified AI)

## OpenRouter MVP Setup

Required:
- OPENROUTER_API_KEY

Optional:
- OPENROUTER_MODEL_A (default: openrouter/free)
- OPENROUTER_MODEL_B (default: openrouter/free)
- OPENROUTER_MODEL_C (default: openrouter/free)
- RETRIEVAL_PROVIDER (mock/web/none, default mock)
- WEB_RETRIEVAL_ENDPOINT
- WEB_RETRIEVAL_API_KEY

`openrouter/free` routes to available free models and may have rate/availability limitations.
